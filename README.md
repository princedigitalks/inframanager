# InfraManager Pro - Next.js UI

Infrastructure Management Console UI built with Next.js 15.

## Run Locally

**Prerequisites:** Node.js 18+

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build for Production

```bash
npm run build
npm start
```

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Motion (Framer Motion)
- Recharts

## Note

This is a UI-only demo with mock data. No backend or API required.
